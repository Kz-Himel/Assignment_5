1️⃣ What is the difference between var, let, and const?  
Ans:- var হলো পুরোনোভাবে ভ্যারিয়েবল ডিক্লেয়ার করার পদ্ধতি এবং এটি function scope। let ও const ES6 এ এসেছে এবং এগুলো block scope। let এর value পরিবর্তন করা যায় কিন্তু const এর value পরিবর্তন করা যায় না।  
2️⃣ What is the spread operator (...)?  
Ans:- pread operator ব্যবহার করা হয় array বা object এর element গুলোকে আলাদা করে ছড়িয়ে দিতে বা copy/merge করতে।  
3️⃣ What is the difference between map(), filter(), and forEach()?  
Ans:- forEach() শুধু array এর প্রতিটি element এর উপর loop চালায়। map() প্রতিটি element পরিবর্তন করে নতুন array তৈরি করে। filter() condition অনুযায়ী কিছু element বেছে নিয়ে নতুন array তৈরি করে।  
4️⃣ What is an arrow function?  
Ans:- Arrow function হলো JavaScript এ ছোট এবং সহজভাবে function লেখার একটি modern syntax যা => ব্যবহার করে লেখা হয়।  
5️⃣ What are template literals?  
Ans:- Template literals হলো backtick ( ) ব্যবহার করে লেখা string যেখানে ${} এর মাধ্যমে variable বা expression সহজে ব্যবহার করা যায়।
