🚀 GitHub Issue Tracker
📖 Overview  
---  
GitHub Issue Tracker is a web-based project that helps developers track bugs, manage feature requests, and organize project tasks efficiently. It uses dynamic API calls to load data in real-time and provides a clean, responsive interface.  
---  
🛠️ Technologies Used  
JavaScript  
JS ES6+
Tailwind CSS  
DaisyUI  
API (for dynamic data)  
✨ Main Features  
🐛 Create, update, and delete issues  
🏷️ Add labels for bug, feature, or improvement  
🔍 Filter and search issues dynamically  
🌐 Fully responsive design for desktop and mobile  
--- 
📦 Dependencies  
{
  "tailwindcss": "^3.x",
  "daisyui": "^4.x"
}
